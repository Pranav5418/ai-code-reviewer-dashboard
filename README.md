# 🚀 AI Code Reviewer Dashboard

A full-stack web application that analyzes Python code and provides real-time feedback using a modern dashboard UI.

---

## 🔧 Tech Stack

- Frontend: React.js
- Backend: FastAPI (Python)
- Visualization: Chart.js
- Static Analysis: Custom complexity + lint analysis

---

## ✨ Features

- 📊 Code Complexity Analysis
- 🧠 Quality Scoring System
- 🧾 Lint Error Detection
- 💡 Smart Code Suggestions
- 📈 Dashboard with Charts
- 📜 History Tracking (Persistent using localStorage)
- ⚡ Real-time Code Analysis
